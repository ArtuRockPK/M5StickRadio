Модули (.h файлы)
📡 wifi_manager.h
wm_loadCreds() — читает SSID/pass из NVS (Preferences)
wm_saveCreds() — сохраняет credentials
runWifiSetup() — блокирующий captive portal:
 • AP «RadioSetup» (open, 192.168.4.1)
 • DNSServer перехватывает все DNS → 192.168.4.1
 • WebServer с формой выбора WiFi
 • Сохраняет creds → ESP.restart()
💾 station_store.h
Объединяет встроенные (stations.h) + пользовательские станции

storeLoad() — загружает из NVS: маску удалённых + custom
getStation(idx) — виртуальный список: сначала не-удалённые built-in, потом custom
storeAdd(name, url) — добавляет custom (max 30)
storeDeleteByIdx() — помечает built-in как удалённую ИЛИ удаляет custom
storeRestoreBuiltins() — сбрасывает маску удалений

NVS namespace: «radio_st»
🌐 web_server.h
WebServer на порту 80 (Core 1)

GET / — HTML страница управления
POST /play, /stop — старт/стоп
POST /next, /prev — переключение ±1
POST /select — прямой переход на станцию по idx
POST /add — добавить станцию
POST /delete — удалить по idx
POST /restore — вернуть встроенные

Все действия → connectStation() или stopPlayback()
🖥 ui.h
initDisplay() — rotation=3, clear
drawUI() — главный экран (по uiDirty):
 • Нормальный режим: название, stream title, счётчик, IP, громкость, спектр
 • Режим громкости: большое число + бар
tickSpectrum() — каждые 80 мс анимирует 8 баров (random targets если isPlaying)
drawConnecting() — экран подключения WiFi
drawAPMode() — экран настройки AP
